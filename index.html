Unmatched Capabilities

A robust plugin architecture designed for absolute context domination.

Hybrid Search (RRF)

Combines exact keyword matches via SQLite FTS5 BM25 with semantic accuracy via local HNSW embeddings, fused together using Reciprocal Rank Fusion.

Modular Plugins

Includes built-in support for Code (tree-sitter parsing), Git histories, and formatted Markdown Docs. Just plug in what you need via @brainbank plugins.

Zero-Config Embeddings

No API keys required out of the box. Computes embeddings completely offline utilizing WASM Transformers for privacy, security, and zero cost inference.

How It Works Under the Hood

1

Facade Orchestrator

brainbank delegates operations to highly-specialized plugin subsystems. The core interacts purely with interfaces, routing operations transparently via a plugin agnostic architectural pattern.

2

AST & Dual Indexing

The code plugin chunker processes scripts generating AST representations leveraging tree-sitter. It then calculates dual embeddings per synopsis file-level and chunk-level simultaneously.

3

Workflow Trace Context

Context generation spans the call graphs. It retrieves recursive hits applying formatting constraints, presenting the LLM with a flawless topographic hierarchy.

example.ts 
import { BrainBank } from 'brainbank';
import { code } from '@brainbank/code';
import { git } from '@brainbank/git';

// Initialize framework and attach capability plugins
const brain = new BrainBank({ repoPath: '.' })
  .use(code())
  .use(git());

// Incrementally update state delta into sqlite
await brain.index();

// Launch complex multi-layered semantic sweeps
const query = 'rate limit routing logic';
const hits = await brain.hybridSearch(query);
