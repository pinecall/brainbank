The @expose decorator marks plugin methods for automatic injection onto the BrainBank instance. After brain.initialize(), exposed methods are callable directly as brain.myMethod().
Collections provide a high-level KV store with built-in hybrid search. Use ctx.collection('name') to get or create one — it handles embedding, HNSW, BM25, and deduplication automatically.
IndexablePlugin makes your plugin run during brain.index(). SearchablePlugin merges results into brain.search() via Reciprocal Rank Fusion. WatchablePlugin auto-re-indexes on file changes.
Factory functions are the recommended export pattern for plugins. Export a function like code(opts) that returns a Plugin instance, keeping the class private and accepting configuration.
